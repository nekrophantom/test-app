## How To Run The Application

### Requirements

Make sure :contentReference[oaicite:0]{index=0} is installed on your computer.

Check Bun version:

```bash
bun --version
```

### 1. Clone Repository

```bash
git clone https://github.com/nekrophantom/test-app.git
```

### 2. Go To Project Folder

```bash
cd test-app
```

### 3. Install Dependencies

```bash
bun install
```

### 4. Run Development Server

```bash
bun run dev
```

### 5. Open In Browser

```txt
http://localhost:3000
```
